# 有效模型声明（CSDE）

## 定位

本文件用于明确当前“胞元基底结构动力学研究引擎”（Cellular-Substrate Structural Dynamics Engine，简称 **CSDE**）的模型边界，避免与标准模型、量子场论、广义相对论和真实实验观测发生口径混淆。

当前相关脚本包括但不限于：

- [hydrogen_color_field_emergence.py](file:///Users/bytedance/TRAEProjects/mind/hydrogen_color_field_emergence.py)
- [particle_family_suite.py](file:///Users/bytedance/TRAEProjects/mind/particle_family_suite.py)
- [cellular_virtual_universe.py](file:///Users/bytedance/TRAEProjects/mind/cellular_virtual_universe.py)

这些脚本统一视为 CSDE 的实验原型或子模块，属于：

- 结构动力学原型
- 低维有效模型
- 无量纲数值实验

它们不是：

- 标准模型的数值求解器
- 量子色动力学的格点实现
- 电弱统一理论的离散化版本
- 广义相对论或宇宙学模型的直接替代

## 一、命名原则

当前项目中出现的下列名称：

- `electron`
- `neutrino`
- `up_quark`
- `down_quark`
- `s-like / p-like / d-like`
- `hydrogen-like`

默认解释为：

- “模式家族标签”
- “结构类比标签”
- “稳定模态的占位命名”

除非另有明确说明，不应解释为：

- 与标准模型粒子一一等价
- 已经复现实验粒子质量谱、电荷谱或散射性质
- 对真实粒子存在性的直接数值证明

因此更推荐的表述是：

- `electron-like mode family`
- `neutrino-like mode family`
- `up-quark-like mode family`
- `hydrogen-like localized structure`

## 二、当前动力学性质

以 [hydrogen_color_field_emergence.py](file:///Users/bytedance/TRAEProjects/mind/hydrogen_color_field_emergence.py) 为例，当前演化由以下结构项组成：

- 离散拉普拉斯扩散
- 自聚焦项
- 饱和排斥项
- 局域 `SU(3)` 生成元旋转项
- 可选角动量通道项

它的目标是研究：

- 是否会从随机场中涌现局域结构
- 是否存在稳定模式窗
- 是否会出现角向节点、模式切换和多尺度谱特征

它当前没有完成的统一内容包括：

- 从单一作用量出发的严格推导
- 与规范场协变导数一致的 `SU(3)` 动力学
- 与洛伦兹不变性一致的传播结构
- 与标准量子统计一致的粒子解释

## 三、守恒量与归一化

当前脚本中常见的“总质量守恒”来自数值归一化步骤，而不是已经证明存在某个严格物理守恒律。

因此应区分：

- 数值上的范数控制
- 物理上的守恒律

当前状态下，可以安全宣称的是：

- 原型通过归一化保持总幅度尺度稳定
- 原型可用于比较不同参数窗下的稳定性和模式差异

当前还不能安全宣称的是：

- 已经构造出真实物理中的能量守恒
- 已经构造出真实物理中的荷守恒
- 已经与标准量子理论中的幺正演化完全一致

## 五、与标准科学理论的关系

当前项目与现有科学理论的推荐关系表述如下：

- 与量子场论：结构类比
- 与 `SU(3)` 色对称：形式对齐
- 与原子轨道：节点形态类比
- 与宇宙学层级：叙事性映射

不推荐的表述：

- “已经证明标准模型错误”

- “原型展示了某类结构动力学是否能产生稳定模式”
- “原型提供了跨尺度锁相与局域模式形成的可视化实验平台”
- “某些模式在形式上与已知物理对象存在类比关系，但尚非物理等价”

## 六、适用域（CSDE）

当前原型适用于：

- 探索局域稳定模式是否存在
- 对比不同控制参数对模式形态的影响
- 生成 2D / 3D 可视化
- 做模式家族之间的结构区分实验

当前原型不适用于：

- 预测真实粒子质量
- 预测真实散射截面
- 代替格点 QCD
- 代替标准宇宙学参数拟合

## 七、闭合球面观测的欠定性（CSDE 观测层）

在当前原型中，闭合球面渲染应理解为一种外显包络或概率壳层的可视化表达，而不应直接理解为对内部动力学自由度的完备观测。该观测层主要保留的是局域分布的外部形态、壳层起伏和随时间演化的概率性外轮廓；但由多胞元、多相位和多中心干涉共同决定的内部结构信息，通常会在包络化、平滑化和投影过程中被压缩、混合或部分消除。

因此，仅凭闭合球面观测，原则上只能恢复某种概率性外形或概率轨迹，而不能唯一恢复内部主轴、真实质心、相位差结构及各子结构分量的瞬时贡献。换言之，在本模型语境下，闭合球面观测对内部状态是欠定的：可能存在多组不同的内部干涉构型，对应到相近甚至近乎不可区分的闭合球面外观。若研究目标涉及轴向偏置、质心漂移、主导通道切换或内部锁相机制，则必须同时引入点云中心、相位编码、密度体素场、主成分轴或其他内部结构观测层，不能仅依赖闭合球面本身做唯一反演。

## 八、后续统一路线（CSDE 演进建议）

若要进一步靠近现行科学理论，建议按以下顺序推进：

1. 给出现有动力学的最小能量泛函或作用量近似
2. 明确哪些项是保守项，哪些项是耗散项
3. 把 `SU(3)` 旋转项重写为更接近规范协变形式
4. 定义可比较的两点函数、功率谱或响应函数
5. 将“粒子”命名统一改写为“模式家族”命名
6. 最后才尝试与标准模型中的对象做一一对应比较

---

## 九、最短结论（CSDE 的定位）

当前 CSDE 原型的最佳定位是：

> 一个零先验、无量纲、以胞元/闭合相/轴谱为原生对象的结构动力学有效模型平台，  
> 用于研究稳定模式、锁相、层级、质量代理与宏观汇聚等现象；  
> 它可以与现有科学理论做形式比较，但尚未与其完成严格统一。
