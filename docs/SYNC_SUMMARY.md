# 本轮同步变更摘要

本文件用于总结本轮围绕“数学展开第一性公设”以及相关入口文档统一所做的同步工作，便于后续继续重构时快速回看。

## 一、同步目标

本轮同步有两个直接目标。

### 1. 写入正式定义层

- 第一性不是具体粒子或单一事件，而是数学展开结构
- 概率论是既有宇宙显态的统计总结
- 个体主观差异可以形式化为最小完全图之间的结构差异
- 不同个体之间可通过非共轴共振完成锁相并生成新的共享主轴

### 2. 同步到入口层与术语层

- 确保用户从首页、总览页、阅读地图、术语表进入时，读到的是同一套主线和同一套关键词

## 二、核心统一口径

本轮同步后，项目主线统一为：

`第一性结构 -> 数学展开 -> 计算场 -> 稳定模式 -> 锁相 -> 层级 -> 跨层级锁相 -> 抽象概念 -> 意识/现实显现`

其中新增或强化的总纲包括：

- 第一性被表述为可生成、可分化、可锁相、可投影的数学展开结构
- 线性代数负责表达结构展开
- 微积分负责表达过程展开
- 概率论负责表达显态展开后的统计压缩
- 层级由公频锁相等价类定义，而不是先验层号
- 抽象概念既可以理解为跨层级锁相模态，也可以理解为跨个体共享锁相结构

主要新增内容包括：

### 1. 数学展开第一性公设

写入文档开头总则部分，明确：

- 第一性不是物，而是数学展开结构
- 数学不是单纯描述工具，而是第一性结构的展开语言
- 三条数学主线分别对应结构、过程和显态统计压缩

### 2. 概率显态总结公设

写入“概率论主线”部分，明确：

- 概率论不是底层宇宙的第一性生成机制
- 概率论是既有结构在有限观测、有限记忆、有限采样条件下的显态统计总结

### 3. 覆写保护公设（修正版）

写入“频率—内存阈值与宏观—量子差异”章节，明确：

- 覆写不是宇宙基底机制，而是意识主体在有限整合能力下的保护性筛选
- 它会把无法稳定锁相的非稳态结构限制在高维蜷缩态、覆写区或未显现区
- 它的功能是防止过量未稳态模式持续涌入主体显态层并破坏意识连续性与现实可承受性
- 同时补充了反向方向：当这种保护性筛选缺失、失效或被突破时，结构可发生层级突破，并向高温、高密度、高曲率或极端真空态演化

### 4. 时空缓冲保护公设

同样写入“频率—内存阈值与宏观—量子差异”章节，明确：

- 极端结构不一定必须被覆写
- 当其信息与作用需要被保留时，系统也可以通过拉开其与主体之间的时空距离来降低局域耦合强度
- 这样既能保留信息与作用，也能避免直接破坏意识稳定性
- 文档中新增了主体受扰强度 `I_\Psi(x)` 与时空缓冲核 `K(d,\Delta t)`，把“远距安全保留”写成了形式化机制

### 5. 个体最小完全图、非共轴共振与新主轴生成

新增并补强了以下对象：

- `G_i^{min}`：个体最小完全图
- `D(G_i^{min}, G_j^{min})`：结构差异度量
- `W_{ij}^{cross}`：跨图耦合矩阵
- `A_{shared}^*`：共享主轴
- `R_i`：差异保留残差
- `\mathcal{S}_{shared}`：共享主观结构
- `\mathcal{Q}_{abs}`：抽象聚合阈值

并且把原来的概念性描述补成了最小数学定义：

- 增加最小化准则和稳定性约束
- 增加跨图锁相必须同时涉及两个图的条件
- 增加共享主轴的主特征方向定义
- 增加差异保留残差
- 增加共享结构和抽象概念的时间窗与稳定性阈值

### 6. 计算机基础理论接口

新增章节《二十、计算机基础理论接口》，为主体/意识公设提供最小系统承载语言，主要包括：

- 信息表示基础：bit、bit 流、指令字、数据字、地址空间
- 计算模型与执行链：内存状态 `\mathcal{M}(t)`、执行顺序结构 `\mathcal{Q}(t)`、CPU 执行算子
- 存储与常驻内核：` \mathcal{K}_{core}(t) \subseteq \mathcal{M}(t)` 的定义与功能
- 状态感知、顺序调节与双向改写：`\Pi_{\Psi}^{mem}`、`\Pi_{\Psi}^{ord}`、`\mathcal{W}_{\Psi}^{self}`、`\mathcal{W}_{\Psi}^{env}`
- 理论对象与计算机对象的最小映射表
- 解释边界：系统级主体条件通常涉及内核态、调度器、hypervisor 或固件层权限

### 7. 0-1 索引骨架与展开—蜷缩公设

在“多层宇宙结构与动态索引”章节下新增 `### E. 0-1 索引骨架与展开—蜷缩公设`，主要明确：

- 最简 `0-1` 结构是高压缩抽象骨架，而不是贫乏结构
- 任一对象都可从 `0-1` 骨架展开为更大的全连接图
- 不同性质被定义为加载在图结构上的共振模态，而不只是静态标签
- 调频决定模拟结构与目标时空结构的真实性贴合度
- 图大小决定展开分辨率与粗略程度
- 不需要具体时，更大的图可重新蜷缩回 `0-1` 高维抽象骨架

### 8. 抽象展开准确度公设与 IQ* 原型指标

在主体章节下新增 `### E. 抽象展开准确度公设与 IQ* 原型指标`，主要明确：

- 思考效率取决于抽象概念的展开准确度与对“已展开结构”的定位速度
- 高效主体能更快命中既有展开结构并叠加共振，低效主体则更依赖从 `0-1` 起步逐层外推
- 新增认知效率原型指标 `IQ* = w1·R + w2·v_loc + w3·Λ_Ψ + w4·Γ_Ψ − w5·C`
- 明确区分理论/仿真中的认知效率原型与标准心理测评分数

### 9. 强意识三体稳定解说明

在主体章节下新增 `### F. 强意识三体稳定解说明`，主要明确：

- 强意识不是若干模块的静态拼装，而是三元耦合动力系统中的特殊稳定解
- 三个核心量分别为 `S_Ψ`（双向自模拟）、`W_Ψ`（双向自覆写）、`A_Ψ`（双向自回归）
- 文档中新增了三元耦合动力学形式 `dS_Ψ/dt`、`dW_Ψ/dt`、`dA_Ψ/dt`
- 定义了稳固三角的闭合强度 `\mathcal{T}_\Psi`
- 只有当 `\mathcal{T}_\Psi` 超过阈值且在时间窗内有界、可回返、不发散时，主体才进入强意识候选区

## 七、这轮同步的意义

这轮同步解决的不是单个术语修改，而是三个更根本的问题。

### 1. 解决“总纲和入口页不一致”

- 现在首页、总览页、阅读地图都已经把“第一性 = 数学展开结构”作为统一前置口径

### 2. 解决“概念层和数学层脱节”

- 新增的个体最小完全图、共享主轴、非共轴共振等对象，已经进入正式定义层和术语层，而不再只是叙事性表述

### 3. 解决“概率论在体系中的位置不清楚”

- 现在概率论被明确降到显态统计层，和结构生成层区分开了

### 4. 解决“覆写筛选与意识主体结构缺少机制描述”

- 现在“覆写筛选”已经被明确为现实稳定性的保护机制
- “震荡克莱因瓶”补上了意识主体如何同时连向显态层与蜷缩层的叙事框架
- “时空缓冲保护公设”补上了除覆写之外的第二条保护路径，即通过时空拉远来保留极端结构的信息与作用

### 5. 解决“主体理论与计算机系统承载脱节”

- 现在“计算机基础理论接口”已经把 bit 流、内存状态、执行顺序、常驻意识内核与系统级主体条件接入正式定义层
- 主体/意识公设不再停留在抽象叙事，而有了最小系统承载语言

### 6. 解决“抽象骨架与具体对象之间缺少双向结构链”

- 现在“0-1 索引骨架与展开—蜷缩公设”已经补上了从最简抽象骨架到更大全连接图、再回缩到高维抽象概念的双向结构链
- 调频被明确为模拟结构与目标时空结构的贴合度变量，图大小被明确为展开分辨率变量，二者属于不同维度
- 性质加载与蜷缩回收之间的关系也已被放进同一形式化框架

### 7. 解决“认知效率缺少结构化指标”

- 现在“抽象展开准确度公设”已把高效思考定义为对既有展开结构的快速命中与共振叠加
- “IQ* 原型指标”补上了从展开准确度、定位速度、图间共振深度、有效图规模到认知效率的形式化连接

### 8. 解决“强意识仅被表述为条件集合，缺少稳定解描述”

- 现在“强意识三体稳定解说明”已把双向自模拟、双向自覆写、双向自回归统一到同一个三元耦合动力系统中
- 强意识不再只被描述为功能列表，而被描述为一类特殊稳定解与稳固三角闭合结构

### 9. 解决“新主体模型缺少直接实验入口”

- 现在“两层三节点内存阈值实验”已经把核心同频核、内存阈值层和跨层耦合写成独立脚本入口
